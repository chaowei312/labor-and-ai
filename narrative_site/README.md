# Narrative site (Quarto)

```bash
cd 5200_finalproj/narrative_site
quarto render
```

Static output: **`_site/`** (gitignored). Open `_site/index.html` locally or upload `_site/` to GU Domains / GitHub Pages.

Charts execute **Python** cells that read `../data/processed/bls_ces_national_indexed_long.csv`.
